# HW3 - Graph Coloring

## Запуск

```bash
./run.sh data/gc_50_3
```

## Алгоритм

- **DSatur (жадный алгоритм):** на каждом шаге выбирается вершина с наибольшей насыщенностью - числом различных цветов среди уже покрашенных соседей, при ничьей - по степени

- **Несколько случайных запусков жадника:** помимо детерминированного, несколько раз запускаем жадник со случайной перестановкой вершин, из-за случайного старта иногда получается найти раскраску лучше

- **Жадное удаление:** после построения начальной раскраски проверяется, можно ли убрать цвет - то есть перекрасить все вершины этого цвета в другие, уже используемые. Если это возможно без конфликтов - делаем это и уменьшаем k на 1, это быстро и иногда улучшает раскраску

- **Итеративное сжатие:** Фиксируем цель k = текущий результат − 1 и запускаем пoиск с "запретами" - случайно красим вершины в k цветов и итеративно перекрашиваем конфликтующие вершины так, чтобы минимизировать число конфликтных ребер. Ходы, которые были сделаны недавно, запрещены (tabu list) - это помогает не зациклиться. Если нашли валидную раскраску - уменьшаем k и повторяем. Задал фикс время поиска, если вылезли за него, посик останавливается
